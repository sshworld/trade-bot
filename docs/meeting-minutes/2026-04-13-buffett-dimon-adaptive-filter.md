# 전문가 토론: 안전장치 → 적응형 필터 전환

**일시**: 2026-04-13
**안건**: 일일 손실 안전장치 유지/수정/제거

## 합의: 적응형 필터 체계

사이즈 절반/강제 중단 → **시그널 품질 필터링**으로 전환.
사이즈는 항상 100% 유지, 진입 기준만 조절.

| 상태 | 트리거 | 최소 strength | 사이즈 |
|------|--------|:---:|:---:|
| **Boost** | 일일 PnL ≥ +3% | 0.50 | 100% |
| **Normal** | 기본 | 0.55 | 100% |
| **Caution** | 일일 PnL ≤ -4% | 0.70 | 100% |
| **Critical** | 일일 PnL ≤ -6% | 0.75 | 100% |
| **Stop** | 일일 PnL ≤ -8% 또는 DD -10% | — | 중단 |

### 변경 사항

| 기존 | 신규 |
|------|------|
| -3% → 사이즈 절반 | **제거** (적응형 필터로 대체) |
| -5% → 당일 중단 | **-8%로 완화** |
| -7% DD → 당일 중단 | **-10%로 완화** |
| 3연속 SL → 30분 중단 | **strength +0.15 (30분간)** |
| 같은 방향 2연속 SL → 30분 | **제거** |

### Boost (잘 되는 날)
- 사이즈 업 기각 (만장일치)
- strength 임계값만 낮춤 (0.55 → 0.50) → 더 많은 기회

**합의 상태**: 전원 동의
