# Buffett × Dimon: 공격적 투자 스타일 토론 (3 Round)
## 2026-04-10

### 개발자 철학
> "매수는 틀려도 분할매수로 평단가 낮추면 되는데, 매도 타이밍 잘 잡는 게 가장 중요"

### Round 1: 개발자 맞나?

**버핏**: "5배 레버리지 선물에서 물타기는 DCA가 아니라 패배에 더블다운. 현재 entry offset 0%/-0.5%/-1.0%의 전체 범위가 SL 거리의 100%. 마지막 tranche가 SL 가격에서 체결됨."

**디먼**: "맞는 말 1가지: 매도 타이밍이 더 중요하다. 고정 TP%는 BTC 변동성에 부적합. 틀린 말: 레버리지 선물에서 물타기는 리스크 증폭기."

### Round 2: 공격적이되 똑똑하게

#### 핵심 전환

| | 보수적 (기존) | 공격적+스마트 |
|---|---|---|
| **진입** | 물타기 (0%, -0.5%, -1%) | **확인 추가** (0%, +0.3%, +0.6%) |
| **청산** | 고정 TP% | **ATR 기반 동적 TP + 트레일링** |
| **TP3** | 고정 목표 | **상한 없음 (트레일링만)** |
| **교체** | 일일 3회 | **일일 4회 + 품질 게이트** |
| **승률 필요** | 83%+ | **35-40%면 충분** (평균 수익 > 2.5× 평균 손실) |

### Round 3: 최종 파라미터

#### 진입 변경
```
WITH_TREND:    50% 즉시 / 30% +0.3% 확인 / 20% +0.6% 확인
COUNTER_TREND: 60% 즉시 / 40% -0.3% (2tranche만)
```
→ 틀리면 50%만 노출. 맞으면 확인 후 추가.

#### ATR 기반 TP/SL
| TF | SL | TP1 | TP2 | TP3 | Split |
|---|---|---|---|---|---|
| 30m | 1.5 ATR | 1.5 ATR | 2.5 ATR | trail | 40/30/30 |
| 1h | 2.0 ATR | 1.5 ATR | 3.0 ATR | trail | 40/30/30 |
| 4h | 2.5 ATR | 2.0 ATR | 4.0 ATR | trail | 40/30/30 |

#### 트레일링 (TP2 체결 후)
- 기본: 최고가 - 2.0×ATR
- 수익 > 5×ATR: 최고가 - 1.0×ATR (조임)
- 하한: TP1 가격 이하로 안 내려감

#### 시간 기반 청산
- 평균 승리 거래 시간의 2배 경과 → SL을 본전+0.5ATR로
- 4배 경과 → 시장가 청산 (이 거래는 안 될 거래)

#### 교체 강화
| | 현재 | 변경 |
|---|---|---|
| 일일 교체 | 3회 | **4회** |
| 쿨다운 | 30분 | **20분** |
| 조건 | PnL < TP1 | **PnL < 0 AND 새 시그널 > 기존+0.5** |
| 같은 시그널 | 5분 쿨다운 | **3분** |
| 일일 거래 | 5회 | **6회** |

### 반드시 유지
- 2% 리스크/거래, 3-5x 레버리지, 동시 1포지션
- 일일 손실 3단계, drawdown halt, anomaly detector
- 지표 패밀리 중복 제거 (이전 회의 합의)

### 핵심 전환
> "자주 지되, 크게 벌어라. 승률 83%에 R:R 1:5보다, 승률 40%에 R:R 2.5:1이 낫다."
