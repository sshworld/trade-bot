# /implement — 기능 구현 자동 루프

$ARGUMENTS 기능을 구현합니다. 검토 → 구현 → 검증을 자동으로 반복합니다.

## 루프

```
1. 계획 수립
   - 변경할 파일 목록
   - 영향 범위 파악

2. 트레이딩 파라미터 변경이 포함되면
   → /trade-review 실행 (전문가 토론 필수)
   → 승인 후 계속

3. 구현
   - 코드 작성/수정
   - 타입 체크 통과 확인

4. 검증
   - Backend: import 확인
   - Frontend: next build 통과
   - API 동작 확인 (curl)

5. 문제 발견 시
   → 3번으로 돌아가서 수정
   → 검증 통과할 때까지 반복

6. 완료
   - 변경 내용 요약 보고
   - CLAUDE.md 업데이트 필요하면 업데이트
```

- 빌드 실패 시 반드시 수정 후 진행
